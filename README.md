# MultiAdapter
针对具有多类型卡片的 RecyclerView 提供的一个不错的解决方案

## 描述
该方案致力于解决 RecyclerView 对多类型卡片的分配问题，让开发者可以更专注于卡片中具体业务的实现。融合了 MultiType、AdapterDelegate、SugarAdapter 等优秀框架的搭建思路，提供了这个较为优雅的解决方案。
